# 大学生竞赛管理系统 java512

## 项目概述

大学生竞赛管理系统是一个基于Springboot和Vue的前后端分离的Java项目。本项目采用MVC三层体系结构，通过整合Spring、SpringBoot、Mybatis、Vue和Maven技术栈进行开发，使用MySQL数据库作为数据存储。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 构建工具：Maven
- 开发架构：MVC

## 功能模块

- 竞赛信息管理
- 学生信息管理
- 赛事报名与分组
- 成绩发布与查询
- 系统用户权限管理

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- 操作系统：兼容性良好，支持Windows、Linux、MacOS等
- 数据库：MySQL
- 服务器：Tomcat
- 前端环境：现代浏览器

## 部署步骤

1. 配置数据库环境
2. 部署后端服务
3. 部署前端服务
4. 启动应用
